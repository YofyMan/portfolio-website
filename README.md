## Visual Design

**Part 2 Questions & Answers**

1. Overall style: Minimalist
2. Color scheme (and why): Blue & White, since they go well with eachother and blue is my favorite color.
3. Fonts (heading / body): sans-serif for the minimalist style
4. How design reflects personality/field: My portfolio uses a clean, modern, and professional design that reflects my interest in computer science and software development. I chose a minimalist layout with consistent colors and typography to keep the focus on my projects and technical skills while maintaining an organized appearance.
5. Homepage layout: The homepage begins with a hero section that introduces me and my career goals, followed by sections for my background, skills, featured projects, and contact information.
6. How project sections are organized visually: Projects are displayed in individual cards with a title, description, and technology icons. This consistent layout makes each project easy to scan while clearly highlighting the purpose of each project and the technologies used.
7. Mobile-friendly approach: The website uses responsive CSS media queries to adjust the navigation, spacing, and layout for smaller screens.
8. Visual hierarchy: The visual hierarchy is created using large headings, section labels, consistent spacing, and contrasting colors. Important information such as my name, career goal, and projects are emphasized first, while supporting details are placed underneath in a logical order.
9. Consistency across pages: Although my portfolio is a single-page website, consistency is maintained through repeated fonts, colors, button styles, spacing, and section layouts.
10. Accessibility considerations: I considered accessibility by using readable fonts, good color contrast, semantic HTML, responsive layouts, and descriptive alt text for images.
11. Icons/images/illustrations used, and why: I included technology icons to visually represent the tools used in each project.
12. Design inspiration (sites/links): My design was inspired by modern developer portfolios that emphasize simplicity, readability, and responsive design.


## Interaction / Functionality

**Part 3 Questions & Answers**

1. Interactive elements included: The website includes a navigation menu, dark mode toggle, contact form, copy email button, and hover effects on buttons and skill tags.
2. Contact form behavior: The contact form validates that all required fields are completed before displaying a confirmation message and clearing the form.
3. JavaScript features implemented: I implemented form validation, a character counter, a copy-to-clipboard button, and a dark mode toggle that saves the user's preference using localStorage.
4. How users receive feedback from interactions: Users receive instant feedback through validation messages, confirmation messages, the live character counter, and notifications when the email is copied.
5. How interactivity improves UX: Interactive features provide immediate feedback, make navigation easier, and create a more engaging and user-friendly experience.
